# CodeAlpha Internship - Python Programming Tasks 

##  Objective
This repository contains Python programming tasks completed as part of the CodeAlpha Internship. The goal is to improve problem-solving, logic building, and Python programming skills.

---

# Task 1: Hangman Game 🎮

## Objective
Guess the hidden word by entering one letter at a time.

## Features
- Random word selection
- Limited wrong attempts (6 chances)
- Tracks correct and wrong guesses
- Win / Lose conditions

---

# Task 2: Stock Portfolio Tracker 📊

## Objective
Calculate total investment based on stock prices and user input.

## Features
- Predefined stock price dictionary
- User inputs stock name and quantity
- Calculates total investment
- Displays final result
- Saves output in a text file (portfolio.txt)

---

# Task 3: Basic Chatbot 🤖

## Objective
Create a simple rule-based chatbot using Python.

## Features
- Responds to "hello"
- Answers "how are you" and "what is your name"
- Ends chat with "bye"
- Handles unknown inputs
